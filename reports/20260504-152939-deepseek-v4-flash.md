2. 不看分数，哪个模型填词结果好？

GPT-5.4 略好，但差距非常小。

实际观察两个模型的填词输出：

词的质量：
- DeepSeek：高频词出现偏多。留守(るす) 出现 13 次，出入り口(でいりぐち) 11 次，代名詞/天井/授業 各 11 次。部分词略显生硬（如
下吏(かり)=下级官吏、伊井(いい)="意大利"的简称）。
- GPT-5.4：用词更多样化，虽然也有高频重复（でも 13 次、忘れ物 10 次），但整体词库覆盖更广。包含一些有趣的词如 辻褄(つじつま)、吃驚(びっくり)、散蒔く(ばらまく)。

跨题多样性：
- DeepSeek：20 个 task 内共 20 个跨题重叠词对（同 task 不同 puzzle 间用相同词）
- GPT-5.4：仅 6 个跨题重叠词对

视觉网格例（task 001 puzzle 0）：

│                                 DeepSeek                                 │                                 GPT-5.4                                  │
├──────────────────────────────────────────────────────────────────────────┼──────────────────────────────────────────────────────────────────────────┤
│ おもい██かく██て██おさらみなもと██い██すんなり██や██だ██くれどういん██つ │ たたく██では██だ██にもつあちこち██か██にぎやか██つ██っ██べつばいてん██れ │

两者都合理填充了整个网格，交叉位置字符一致。GPT-5.4 用词（叩く/賑やか/荷物/売店）在 crossword 语境下感觉更自然，DeepSeek（重い/すんなり/動員）稍逊。

结论：GPT-5.4 的词质量稍好、多样性略高，但差距在 5% 以内，非专业用户不易察觉。

3. 分数和指标能否反映结果好坏？

能，但有两处需要注意。

分数对比

┌────────────────────────┬──────────┬──────────┬────────────┐
│          指标          │ DeepSeek │ GPT-5.4  │    差值    │

│ avg overallScore       │ 0.9982   │ 0.9989   │ GPT +0.07% │

│ avg validPuzzleRate    │ 1.0000   │ 1.0000   │ 持平       │
├────────────────────────┼──────────┼──────────┼────────────┤

├────────────────────────┼──────────┼──────────┼────────────┤
│ avg crossPuzzleVariety │ 0.9928   │ 0.9955   │ GPT +0.27% │
├────────────────────────┼──────────┼──────────┼────────────┤
│ avg finalScore         │ 0.9948   │ 0.8116   │ DS +22.6%  │

│ avg elapsedMs          │ 174ms    │ 16,575ms │ DS 快 95x  │
└────────────────────────┴──────────┴──────────┴────────────┘

分数反应准确的地方：
- validPuzzleRate 都是 1.0 → 两模型都能稳定输出合法题目 ✓
- preferenceFit 都是 1.0 → 都能完美满足软约束 ✓
- crossPuzzleVariety GPT-5.4 略高 → 实际查看确实 GPT 跨题重复更少 ✓

分数不完美的地方：

1. overallScore 的区分度太低。两个模型的 overallScore 都在 0.998+，实际差距 0.07%，几乎没区分度。这是因为 (a) validPuzzleRate 和 preferenceFit 都封顶 1.0（占 75%
权重），(b) crossPuzzleVariety 的公式 1 - 0.7*avgPairwiseOverlap - 0.3*globalReuseRate 让得分普遍偏高。仔细看：DeepSeek 的 averagePairwiseOverlap 最高
0.011（page-008），GPT 最高 0.022（page-006），但 1 - 0.7*0.022 - 0.3*0.044 = 0.97，仍有高分。
2. 分数没有反映"词的自然度"。比如 DeepSeek 用了 下吏(かり)（罕见词），GPT 用了
吃驚(びっくり)（更自然），但分数上没有任何体现。这是当前评测体系的一个盲区——它只检查"词在词库里"，不评"词是否适合 crossword"。
3. finalScore 的时间惩罚对 GPT 过重。GPT 慢 95 倍，时间分只有 0.06，导致 finalScore 从 0.9989 降到 0.8116。时间对 crossword
生成来说并不是核心指标（用户不介意等几十秒），finalScore 的 20% 时间权重 + 相对时间分机制，在这里让 GPT 的排名明显低于实际体验。

总体评价：分数体系能有效区分"能不能用"（validPuzzleRate），但难以区分"好不好用"（词自然度、做题体验）。当前分数下两个模型都是优秀的，DeepSeek 以压倒性的速度优势取得
finalScore 胜利，但实际填字结果质量 GPT-5.4 稍优。
